# U05ClientDemo
此为康力优蓝优友机器人U05客户端定制demo源码<br> 
<br>   目前提供的功能有：
   语音唤醒、
   语音识别、
   对接客户的语义系统（知识库）、
   机器人动作、
   眼睛表情、
   播放音频、
   播放TTS、
   地图导航、
   人脸追踪、
   人脸识别、
   感应器、
   人体感应等
   
   用户在机器人端定制开发APP请参考此 demo 以及 [机器人开发文档](https://www.kancloud.cn/jessie_lee/doc1/705465)
